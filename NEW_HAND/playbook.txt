階段一：VS Code 基礎載入與指甲識別
目標： 在 OpenGL 視窗中成功顯示手部，並讓指甲與皮膚分開。

專案骨架與環境設定 (ID 1.1)：

在 VS Code 中配置 OpenGL / GLFW 專案 。

引入 Assimp 或 tinyobjloader 函式庫來讀取你的 .obj 檔案 。

載入模型與紋理 (ID 1.2)：

撰寫 Model 類別，讀取剛從 Blender 匯出的 .obj 檔案 。

載入 textures/ 資料夾下的皮膚貼圖 。

材質區分邏輯 (ID 1.3)：

在渲染循環中，偵測材質名稱 。


關鍵動作：如果材質名稱為 Nails，將一個 uniform int isNail 設為 1，否則設為 0 。

時間參數同步 (ID 1.4)：

在主迴圈獲取 glfwGetTime()，傳遞給 Shader 中的 uniform float time 。

階段二：Geometry Shader 基礎設置
目標： 建立幾何著色器管線，並讓指甲表面產生動態。

建立 manicure.geom (ID 2.1)：

設定輸入為 triangles 。

設定輸出為 triangle_strip，最大頂點數視裝飾複雜度而定（建議先設 12） 。

Interface Block 傳接 (ID 2.2)：

使用 gs_in[] 接收來自 Vertex Shader 的法線（Normal）與 UV 座標 。

計算表面法向量 (ID 2.3)：

實作 GetNormal() 函式，利用 cross() 計算三角形的法線方向 。

實作 Explode 位移 (ID 2.4)：

撰寫 explode() 函式，當 isNail == 1 時，利用 time 與 normal 讓三角形沿著法線往外推移 。

階段三：高階幾何生長動畫
目標： 在指甲上「生長」出立體飾品（如錐體或藤蔓）。

飾品生成演算法 (ID 3.2)：

在 Geometry Shader 的 main() 函式中，除了輸出原本的三角形，額外增加一組頂點。


實作方式：根據三角形中心點，沿著法線延伸出一個頂點，並與原頂點連成一個三角錐 。

動態生長動畫 (ID 3.3)：

將延伸出的頂點長度乘以 sin(time)，造成飾品不斷長出又縮回的視覺效果 。

視覺豐富度優化 (ID 3.4)：

在 Fragment Shader 中，針對飾品（幾何著色器產生的新頂點）加入強烈的高光（Specular）效果，模擬鑽石或金屬質感 。
